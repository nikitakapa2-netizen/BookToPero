# Лист и Перо — интернет-магазин книг

Готовый проект курсовой работы на PHP 8 + Apache + MySQL (OpenServer).

## Запуск
1. Скопируйте проект в `OpenServer/domains/BookToPero`.
2. Создайте домен (например, `booktopero.local`) и направьте на папку проекта.
3. Создайте БД `bookstore` (или просто выполните SQL целиком).
4. Импортируйте `sql/bookstore.sql`.
5. Откройте `includes/config.php` и при необходимости укажите параметры БД/SMTP.
6. Запустите OpenServer и откройте сайт в браузере.

## Тестовые аккаунты
- Администратор: `admin / admin123`
- Пользователь: `user / user123`

## Структура
- Публичная часть: главная, каталог, карточка книги, корзина, заказ, профиль.
- Админка: книги (CRUD), категории, заказы (поиск/фильтр/статусы), пользователи, контакты.
- Слои: `includes/db.php` (PDO), `includes/queries.php` (запросы), `includes/auth.php` (доступ), `includes/functions.php` (утилиты).

## Почта
После оформления заказа вызывается `includes/mail.php`.
- При `SMTP_ENABLED=false` письмо не отправляется во внешний SMTP, а сохраняется в `storage/mail.log` (fallback).
- Для SMTP: установите PHPMailer через Composer и настройте `SMTP_*` в `includes/config.php`.

## Примечание
Проект содержит демо-данные: роли, категории, книги, отзывы, статусы заказов и пользователей.
