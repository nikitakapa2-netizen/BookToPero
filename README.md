# Лист и Перо — интернет-магазин книг

Готовый проект курсовой работы на PHP 8 + Apache + MySQL (OpenServer).

## Запуск
1. Скопируйте проект в `OpenServer/domains/BookToPero`.
2. Создайте домен (например, `booktopero.local`) и направьте на папку проекта.
3. Создайте БД `bookstore`.
4. Импортируйте `sql/bookstore.sql`.
5. При необходимости измените параметры БД/SMTP в `includes/config.php`.
6. Запустите OpenServer и откройте сайт.

## Тестовые аккаунты
- Администратор: `admin / admin123`
- Пользователь: `user / user123`

## Новые возможности
- Главная страница с большим левым фильтром (доступен, новинка, лидер продаж, скидки, самовывоз и расширенные параметры).
- Разделы каталога, фильтрация по цене/автору/издательству/году/переплету/бумаге/языку.
- Список желаемого (wishlist), сердечки на карточках и отдельная страница.
- Блок «Будущие книги» (скоро в продаже).
- Пагинация списка книг на главной.
- Автослайдер отзывов.
- Кнопка «вверх» справа.
- Отдельные страницы: «Помощь», «Способы оплаты», и красивая `404`.
- В checkout для самовывоза добавлен выбор пункта выдачи.
- В заказах добавлен чат между админом и пользователем.
- В админке у пользователей добавлена смена пароля.
- В форме редактирования книг добавлен превью изображения обложки.
